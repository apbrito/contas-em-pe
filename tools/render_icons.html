🦉 Gerar ícones PWA

Esta página renderiza o ícone SVG (icons/icon.svg) em três tamanhos PNG necessários ao PWA. Corre uma vez, descarrega os 3 ficheiros, e mete-os em icons/.

  1. Verifica que os 3 previews abaixo mostram o mocho 🦉 sobre fundo bege.
  2. Carrega em Descarregar todos.
  3. Copia os ficheiros para a pasta icons/ do projecto.
  4. Faz git add icons/*.png e commit.
